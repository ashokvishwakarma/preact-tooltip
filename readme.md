# Tooltip for Preact

A tool-tip component for `preact`.

## Install

```
npm install --save tooltip-preact
```

## Uses

```
import Tooltip from 'tooltip-preact';

render() {
  return (
    <div>
      Some text
      <Tooltip label="Tooltip text" className="custom-class">
    </div>
  )
}
```

## Options

* label - the tooltip text
* className - additional class name
